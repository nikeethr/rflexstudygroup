Conjecture: $\forall n \in \mathcal{Z}\,[5|n + n+1 + n+2 + n+3 + n+4]$.

For an arbitrary $n$,
\begin{align*}
  n + n + 1 + n + 2 + n + 3 + n + 4 = 5n + 10 = 5(n + 2) && \text{[basic algebra]}
\end{align*}
which is clearly divisibly by 5. Therefore, the conjecture is true.
